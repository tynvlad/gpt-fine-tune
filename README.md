Fine-tuning GPT

Node.js
Python
OpenAi Cli (pip install --upgrade openai openai"[datalib]")

```bash
>> pip install openai
>> pip install openai[datalib]
```

change data

```bash
>> openai tools fine_tunes.prepare_data -f ./data.jsonl
```
